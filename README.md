# Education4GenZ

**Education4GenZ** is a simple educational blog website designed for students and Gen Z learners.
The project follows a traditional **content-rich blogging structure** with multiple articles, categories, archives, and educational resources.

The goal of this project is to build a **classic multi-page blog** that looks like a long-running educational website filled with useful content for students.

---

## 🌐 Website

https://atulg-spec.github.io/Education4GenZ/)

---

## 📚 About the Project

Education4GenZ focuses on providing helpful educational content such as:

* Study tips for students
* Exam preparation strategies
* Career guidance
* Digital skills
* Online learning resources
* Productivity techniques
* Programming and technology basics

The website is designed to feel like a **traditional blog that has been active for years**, with lots of articles and information.

---

## 🛠 Tech Stack

The project is intentionally kept simple and lightweight.

* **HTML**
* **Tailwind CSS (CDN)**
* **Vanilla JavaScript**

No frameworks, no build tools, and no complex dependencies.

---

## ✨ Features

* Multi-page blog structure
* Content-heavy homepage with many articles
* Individual blog post pages
* Sidebar with search, categories, and archives
* Tag cloud for topics
* Newsletter subscription UI
* Comment section UI (frontend only)
* Simple responsive design using Tailwind CSS
* Mobile navigation toggle with JavaScript
